    gap

    @function gap($top: null, $right: null, $bottom: null, $left: null, $font-size: null, $unit: null) { ... }

    Description

    This function arrange margin and padding property values of component use by font-size property. Another benefit rescale component size relatively of component's font-size property value.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $top

    Top value of property.

    		Unitless or Stringnull
    $right

    Right value of property.

    		Unitless or Stringnull
    $bottom

    Bottom value of property.

    		Unitless or Stringnull
    $left

    Left value of property.

    		Unitless or Stringnull
    $font-size

    font-size value of component.

    		Unitless or Stringnull
    $unit

    If want to use another value sould be pass in this variable.

    		Stringnull

    Example

    gap function

    .deneme {
 margin: gap(20px, 30px, 40px, 60px, 10px, rem);
}

/**
 .deneme {
  margin: 2rem 3rem 4rem 6rem;
}
*/

    Requires

    • [function] strip-unit
    • [function] strip-unit

    devices

    @mixin devices($breakpoint: $breakpoint-list) { ... }

    Description

    This mixin will be create responsive layout for page or components use by media at-rule.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $breakpoint

    Reference of breakpoint component. If $breakpoint without use then this mixin will be generate all breakpoints depend of $breakpoint-list. If want to numerical value for breakpoint, value suffix must be px.

    		List or String$breakpoint-list

    Example

    devices mixin

    .page {
  background-color: red;
  /** When parameter without use then this mixin will be generate for all breakpoints depends of $breakpoint variable values. */
  @include devices(mobile) {
    background-color: green;
  }
}

/**
.page {
 background-color: red;

media only screen and (588px > width) {
 .page {
   background-color: green;
 }
 */

    component-query

    @mixin component-query($breakpoint: $breakpoint-list) { ... }

    Description

    This mixin will be generate component style based on width of component.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $breakpoint

    Reference of breakpoint component. If $breakpoint without use then this mixin will be generate all breakpoints depend of $breakpoint-list. If want to numerical value for breakpoint, value suffix must be px.

    		List or String$breakpoint-list

    Example

    component-query mixin

    .component {
  background-color: red;
    /** When parameter without use then this mixin will be generate for all breakpoints depends of $breakpoint variable values. */
  @include component-query(mobile) {
    display: grid;
  }
}

/**
 .component {
  background-color: red;
}
 @container mobile 588px>width {
  .component {
    display: grid;
  }
}
 */
